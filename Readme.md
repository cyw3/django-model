# django通用模板，
主要针对前端插件，界面进行模板整理。
主要目录有static和templates

插件包括：
jquery，fontawesome这些基础插件外：
表单下拉框样式
chosen
表单颜色选择
colorpalette
colorpicker
表单日期选择
datepicker

滚动条样式
mmenu
nicescroll

parsley  #整合了前端验证

animat-number  数字动画


hchart--图表生成
tinymce--文本编辑器
editor.md--markdown编辑器
并自带模板的样式

cookie   抓取cookie
